# 随机生成器工作台（公开版）

一个可直接在浏览器中打开的本地前端工作台，用来生成人物背景、性格档案与人物外观（默认关闭 NSFW 设置）。

## 功能概览

- 人物背景：城市、成长地区、教育与职业、关系状态、生活节奏
- 性格档案：核心维度、MBTI 派生、人际面向与桥接层
- 人物外观：整体轮廓、面部结构、口腔、腹臀腿、手足等模块
- 支持模块重刷、字段手选、卡片锁定、串联生成与 JSON 导出
- 私密部位与偏好档案仍然保留在工作台里，但公开版默认不展开

## 使用方式

- 直接打开 `index.html`
- 在顶部标签页中切换不同生成器
- 点击 `串联生成` 可以按当前公开链路快速生成一位角色
- 如需补充私密设定，可点击顶部 `设定私密部位与偏好`

## 目录说明

- `index.html`：页面入口
- `generator_workspace.css`：统一样式
- `generator_workspace_ui.js`：顶部交互、卡片工具与导出逻辑
- `*_workspace.js` / `*_data.js`：对应生成器逻辑与本地数据
